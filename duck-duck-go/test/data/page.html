Use DuckDuckGo for Chrome search?

OK
  1. Click OK button above to add DuckDuckGo.
  2. Click the settings/hamburger icon (≡) on the Chrome toolbar (top right).
  3. Select Settings from the drop-down menu.
  4. Under Search section, click Manage search engines...
  5. Find DuckDuckGo in the displayed list and click Make Default.

Set DuckDuckGo as your start page?

  1. Click settings/hamburger icon (≡) on the Chrome toolbar (top right).
  2. Select Settings from the drop-down menu.
  3. Under On startup, click Open a specific page then click Set pages.
  4. Click Use current pages then Click OK.
